If you want runnable code instead of isolated snippets, start here.

## Repository examples

- [`packages/protobuf-example`](https://github.com/bufbuild/protobuf-es/tree/main/packages/protobuf-example): A small application that uses generated Protobuf types.
- [`packages/protoplugin-example`](https://github.com/bufbuild/protobuf-es/tree/main/packages/protoplugin-example): A custom plugin that generates Twirp clients and uses custom options.

## Related projects

- [Connect-ES](https://github.com/connectrpc/connect-es): RPC support built on Protobuf-ES service descriptors.
- [Protovalidate-ES](https://github.com/bufbuild/protovalidate-es): Validation for Protobuf messages, with a [Valid types example](https://github.com/bufbuild/protovalidate-es/tree/main/packages/example#valid-types).
